# 📡 maro-plugin-http

Hace hablar a las APIs y desvela su esencia.

Con este plugin, Maro puede generar automáticamente documentación Swagger y colecciones Postman a partir de archivos `.http`, convirtiendo especificación en exploración instantánea.

Descubre, documenta y prueba tus endpoints sin salir de la brújula del terminal.

---
**Comandos principales**
- Generar documentación Swagger desde archivos `.http`.
- Crear colecciones Postman listas para testear.
- Traducir tus flujos de API en puentes tangibles para equipos y automatizaciones.

Conéctate sin fricción con el plano invisible de tus servicios web.
